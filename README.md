# Svitky Hrdinů – Foundry VTT Content Module

> Neoficiální content modul pro Foundry VTT se svolením autora hry.

**Svitky Hrdinů** jsou původní česká fantasy RPG hra postavená na enginu Powered by the Apocalypse. Oficiální stránka hry: [svitky.nord.ninja](https://svitky.nord.ninja/)

---

## Požadavky

| Závislost | Odkaz |
|---|---|
| Foundry VTT | v14+ |
| Systém **PbtA** (povinný) | [foundryvtt.com/packages/pbta](https://foundryvtt.com/packages/pbta) |
| Modul **Party Resources** (doporučený pro trackování Situace) | [foundryvtt.com/packages/fvtt-party-resources](https://foundryvtt.com/packages/fvtt-party-resources) |

---

## Obsah modulu

Systém nepodporuje hexy a škrty a nahrazuje je běžnými číselnými počítadly, které teda bude třeba násobit třemi (6 Vitality namísto 2 hexů se třemi škrty).

### Deníky postav

Předkonfigurované šablony deníků pro:

- **Hráčské postavy**
- **Nehráčské postavy (NPC)**
- **Společníky**

### Kompendium: Svitky Hrdinů

Kompletní herní obsah hry:

- **Základní tahy** 
- **Dovednosti povolání, rodů a společníků**
- **Kouzla**
- **Předměty** – veškeré předměty z obecného inventáře a startovního vybavení svitků povolání a rodů a předměty vázané na dovednosti povolání a rodů (e. g. alchymistické předměty, kosti předka)

### Kompendium: Makra

Sada maker pro zrychlení práce Průvodce i hráčů:

| Makro | Popis |
|---|---|
| **Hromadný import dovedností a předmětů** | Umožňuje rychlý import většího množství dovedností a předmětů |
| **Generátor tabulky** | Vytvoří tabulku ve formátu `1–1 text, 2–2 text, ...` |
| **Přepínač vlastností Měňavce** | Přepíná vlastnosti postavy Měňavce mezi přirozenou a proměněnou podobou |
